# 🧩 模块说明

- **cmd/**：应用启动入口（main.go）
- **internal/app/**：组合所有服务逻辑
- **internal/config/**：配置加载
- **internal/common/**：公共
- **internal/base/**：基础服务
- **internal/server/**：Gin/gRPC 启动逻辑
- **internal/middleware/**：中间件
- **internal/handler/**：HTTP 控制器
- **internal/model/**：数据模型
- **internal/repository/**：数据库操作
- **internal/service/**：业务逻辑
- **pkg/**：可复用库
